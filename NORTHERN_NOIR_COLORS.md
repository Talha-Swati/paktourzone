# Northern Noir Color Reference

## Use these exact values in components:

```css
/* Backgrounds */
background: #0B0C0E;  /* Carbon Black - Main BG */
background: #0F1A27;  /* Midnight Navy - Sections */
background: #141A1F;  /* Deep Slate - Cards */

/* Borders */
border-color: #1E242B;  /* Charcoal Smoke */
border-color: #4DBBFF;  /* Ice Blue - Hover */

/* Text */
color: #F2F6F9;  /* Glacier White - Headings */
color: #C4CCD4;  /* Moon Mist - Body */

/* Accents */
color: #4DBBFF;  /* Ice Blue */
background: #22D3EE;  /* Aurora Cyan - Buttons */
background: #7EE7FD;  /* Light Cyan Aura - Hover */

/* Gradients */
linear-gradient(135deg, #4DBBFF 0%, #22D3EE 100%)  /* Northern Lake */
linear-gradient(180deg, #0B0C0E 0%, #0F1A27 100%)  /* Night Sky */
```
